# HTML,CSS基礎 受講者配布版

この課題では、デザイン見本をもとに **HTML / CSSのみ** でプロフィールカード一覧ページを実装します。

## 課題の目的
- HTMLの基本構造を理解する
- セマンティックタグを使って画面構造を作る
- CSSでレイアウト・余白・見た目を整える
- Flexboxで横並びを実装する
- media queryでレスポンシブ対応を行う

## 成果物
以下の見た目を参考に、プロフィールカード一覧ページを完成させてください。

- ヘッダー
- メインコンテンツ
- プロフィールカード3件
- フッター
- PCでは3列表示
- スマートフォンでは1列表示

## ファイル構成
- `index.html`
  - HTMLを記述するファイル
- `style.css`
  - CSSを記述するファイル
- `reference/`
  - 参考用の完成版UI
  - まずは見ずに自力で取り組み、最後の比較確認に使ってください

## 実装条件
- HTML / CSSのみで実装する
- JavaScriptは使用しない
- Flexboxを使ってカード一覧を横並びにする
- ボタンにhover時の見た目変化を付ける
- 768px以下で1列表示にする

## 作業の進め方
1. `index.html` に画面の骨組みを作る
2. `style.css` で全体の見た目を整える
3. カード一覧を3列にする
4. hoverスタイルを追加する
5. レスポンシブ対応を追加する
6. 完成版と見比べて差分を確認する
